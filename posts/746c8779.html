е…ій”®и®ҫи®ЎеҶізӯ–пјҡдёәд»Җд№Ҳз”Ё LLM иҖҢдёҚжҳҜдј з»ҹзҡ„ NERпјҲе‘ҪеҗҚе®һдҪ“иҜҶеҲ«пјүе·Ҙе…·пјҹ

дј з»ҹ NER е·Ҙе…·пјҲеҰӮ spaCyгҖҒStanford NERпјүж“…й•ҝиҜҶеҲ«вҖқдәәеҗҚгҖҒең°еҗҚгҖҒз»„з»ҮеҗҚвҖқиҝҷзұ»йҖҡз”Ёе®һдҪ“пјҢдҪҶеңЁдё“дёҡйўҶеҹҹпјҲжҠҖжңҜж–ҮжЎЈгҖҒжі•еҫӢеҗҲеҗҢгҖҒеҢ»еӯҰж–ҮзҢ®пјүдёӯиЎЁзҺ°еҫҲе·®гҖӮLLM зҡ„дјҳеҠҝеңЁдәҺйӣ¶ж ·жң¬жіӣеҢ–вҖ”вҖ”дҪ еҸӘйңҖиҰҒеңЁ prompt дёӯе®ҡд№үе®һдҪ“зұ»еһӢпјҢе®ғе°ұиғҪжҸҗеҸ–пјҢдёҚйңҖиҰҒж ҮжіЁж•°жҚ®и®ӯз»ғгҖӮ

еҪ“з„¶пјҢд»Јд»·жҳҜжҲҗжң¬гҖӮжҜҸеӨ„зҗҶдёҖдёӘж–ҮжЎЈеқ—йғҪиҰҒи°ғз”ЁдёҖж¬Ў LLM APIпјҢеҜ№дәҺеӨ§и§„жЁЎиҜӯж–ҷжқҘиҜҙпјҢзҙўеј•жҲҗжң¬еҸҜиғҪйқһеёёй«ҳгҖӮеҗҺйқўжҲ‘д»¬дјҡи®Ёи®әеҰӮдҪ•дјҳеҢ–гҖӮ

Step 2пјҡе®һдҪ“ж¶Ҳжӯ§дёҺеҗҲе№¶

дёҚеҗҢж–ҮжЎЈеқ—еҸҜиғҪз”ЁдёҚеҗҢзҡ„еҗҚеӯ—жҢҮд»ЈеҗҢдёҖдёӘе®һдҪ“гҖӮжҜ”еҰӮвҖқSpring BootвҖқгҖҒвҖқSpringBootвҖқгҖҒвҖқSBвҖқеҸҜиғҪжҳҜеҗҢдёҖдёӘдёңиҘҝгҖӮGraphRAG з”Ё LLM еҒҡе®һдҪ“ж¶Ҳжӯ§пјҡ

иҝҷдёҖжӯҘеҜ№жңҖз»Ҳеӣҫи°ұиҙЁйҮҸеҪұе“Қе·ЁеӨ§гҖӮеҰӮжһңж¶Ҳжӯ§еҒҡеҫ—дёҚеҘҪпјҢеҗҢдёҖдёӘе®һдҪ“иў«жӢҶжҲҗеӨҡдёӘиҠӮзӮ№пјҢеӣҫи°ұзҡ„иҝһйҖҡжҖ§е°ұдјҡдёӢйҷҚпјҢзӨҫеҢәжЈҖжөӢзҡ„ж•Ҳжһңд№ҹдјҡжү“жҠҳжүЈгҖӮ

Step 3пјҡжһ„е»әзҹҘиҜҶеӣҫи°ұ

жҠҠжҸҗеҸ–еҲ°зҡ„е®һдҪ“дҪңдёәиҠӮзӮ№гҖҒе…ізі»дҪңдёәиҫ№пјҢжһ„е»әдёҖеј еӣҫгҖӮGraphRAG зҡ„е®һзҺ°дёӯдҪҝз”Ёзҡ„жҳҜNetworkXпјҲPython еӣҫи®Ўз®—еә“пјүпјҢеңЁеҶ…еӯҳдёӯз»ҙжҠӨеӣҫз»“жһ„гҖӮ

Step 4пјҡзӨҫеҢәжЈҖжөӢдёҺеұӮзә§ж‘ҳиҰҒ

иҝҷжҳҜ GraphRAG жңҖж ёеҝғзҡ„еҲӣж–°гҖӮ

зҹҘиҜҶеӣҫи°ұе»әеҘҪеҗҺпјҢGraphRAG еҜ№еӣҫеҒҡLeiden зӨҫеҢәжЈҖжөӢз®—жі•гҖӮзӨҫеҢәжЈҖжөӢзҡ„зӣ®зҡ„жҳҜжҠҠеӣҫдёӯзҙ§еҜҶиҝһжҺҘзҡ„иҠӮзӮ№еҲҶжҲҗдёҖз»„вҖ”вҖ”жҜҸз»„е°ұжҳҜдёҖдёӘвҖқзӨҫеҢәвҖқпјҢд»ЈиЎЁдёҖдёӘдё»йўҳжҲ–зҹҘиҜҶз°ҮгҖӮ

з„¶еҗҺпјҢGraphRAG еҜ№жҜҸдёӘзӨҫеҢәз”Ё LLM з”ҹжҲҗж‘ҳиҰҒпјҡ

жӣҙеҰҷзҡ„жҳҜпјҢLeiden з®—жі•жҳҜеұӮзә§еҢ–зҡ„вҖ”вҖ”е®ғеҸҜд»ҘеңЁдёҚеҗҢзІ’еәҰдёҠеҒҡзӨҫеҢәжЈҖжөӢгҖӮеә•еұӮзӨҫеҢәжҳҜе°ҸиҖҢзІҫзҡ„дё»йўҳз°ҮпјҢдёҠеұӮзӨҫеҢәжҳҜжӣҙеӨ§зҡ„зҹҘиҜҶеҹҹгҖӮGraphRAG дёәжҜҸдёҖеұӮйғҪз”ҹжҲҗж‘ҳиҰҒпјҢеҪўжҲҗеұӮзә§еҢ–зҡ„зҹҘиҜҶзҙўеј•гҖӮ

иҝҷдёӘеұӮзә§з»“жһ„зҡ„ж„Ҹд№үеңЁдәҺпјҡдёҚеҗҢзІ’еәҰзҡ„й—®йўҳеҸҜд»Ҙе‘ҪдёӯдёҚеҗҢеұӮзә§зҡ„зӨҫеҢәж‘ҳиҰҒгҖӮ

жҹҘиҜўйҳ¶ж®өпјҡеұҖйғЁжҗңзҙў vs е…ЁеұҖжҗңзҙў

GraphRAG еңЁжҹҘиҜўж—¶жҸҗдҫӣдёӨз§ҚжҗңзҙўжЁЎејҸпјҢеҲҶеҲ«еә”еҜ№дёҚеҗҢзұ»еһӢзҡ„й—®йўҳгҖӮ

еұҖйғЁжҗңзҙўпјҲLocal Searchпјү

йҖӮеҗҲзІҫзЎ®жҹҘжүҫзұ»й—®йўҳвҖ”вҖ”и·ҹдј з»ҹ RAG зұ»дјјпјҢдҪҶеҲ©з”ЁдәҶеӣҫи°ұзҡ„з»“жһ„дҝЎжҒҜгҖӮ

е·ҘдҪңжөҒзЁӢпјҡ

  1. е®һдҪ“иҜҶеҲ«пјҡд»Һз”ЁжҲ·й—®йўҳдёӯжҸҗеҸ–е®һдҪ“пјҲеҰӮвҖқSpring BootвҖқпјү
  2. еӣҫи°ұйҒҚеҺҶпјҡд»ҺиҜҘе®һдҪ“иҠӮзӮ№еҮәеҸ‘пјҢжІҝиҫ№йҒҚеҺҶ 1-2 и·іпјҢж”¶йӣҶе…іиҒ”е®һдҪ“е’Ңе…ізі»
  3. ж–ҮжЎЈжЈҖзҙўпјҡеҗҢж—¶з”Ёеҗ‘йҮҸжЈҖзҙўжүҫеҲ°зӣёе…іж–ҮжЎЈеқ—
  4. дёҠдёӢж–Үз»„иЈ…пјҡжҠҠеӣҫи°ұдҝЎжҒҜпјҲе®һдҪ“гҖҒе…ізі»гҖҒзӨҫеҢәж‘ҳиҰҒпјүе’Ңж–ҮжЎЈеқ—жӢјжҲҗ prompt
  5. LLM з”ҹжҲҗпјҡеҹәдәҺз»„иЈ…еҘҪзҡ„дёҠдёӢж–Үеӣһзӯ”й—®йўҳ

еұҖйғЁжҗңзҙўзҡ„дјҳеҠҝеңЁдәҺзІҫзЎ®жҖ§вҖ”вҖ”е®ғеҲ©з”Ёеӣҫи°ұзҡ„з»“жһ„е…ізі»жқҘиЎҘе……еҗ‘йҮҸжЈҖзҙўеҸҜиғҪйҒ—жјҸзҡ„дёҠдёӢж–ҮгҖӮ

е…ЁеұҖжҗңзҙўпјҲGlobal Searchпјү

йҖӮеҗҲе…ЁеұҖжҖ§жҹҘиҜўвҖ”вҖ”иҝҷжҳҜ GraphRAG зңҹжӯЈзҡ„жқҖжүӢй”ҸгҖӮ

е·ҘдҪңжөҒзЁӢпјҡ

  1. йҖүжӢ©зӨҫеҢәеұӮзә§пјҡж №жҚ®й—®йўҳзҡ„зІ’еәҰйҖүжӢ©еҗҲйҖӮзҡ„зӨҫеҢәеұӮзә§
  2. е№¶иЎҢж‘ҳиҰҒпјҡеҜ№жҜҸдёӘзӨҫеҢәзҡ„ж‘ҳиҰҒпјҢз”Ё LLM з”ҹжҲҗдёҖдёӘдёӯй—ҙзӯ”жЎҲ
  3. Map-Reduce иҒҡеҗҲпјҡжҠҠжүҖжңүдёӯй—ҙзӯ”жЎҲжұҮжҖ»пјҢз”ҹжҲҗжңҖз»Ҳеӣһзӯ”

иҝҷдёӘиҝҮзЁӢжң¬иҙЁдёҠжҳҜдёҖдёӘ Map-Reduce жЁЎејҸпјҡ

дёәд»Җд№Ҳиҝҷж ·и®ҫи®Ўпјҹ еӣ дёә LLM зҡ„дёҠдёӢж–ҮзӘ—еҸЈжҳҜжңүйҷҗзҡ„гҖӮдҪ дёҚеҸҜиғҪжҠҠж•ҙдёӘж–ҮжЎЈеә“зҡ„жүҖжңүеҶ…е®№дёҖж¬ЎжҖ§еЎһиҝӣеҺ»гҖӮдҪҶйҖҡиҝҮзӨҫеҢәж‘ҳиҰҒзҡ„еұӮзә§з»“жһ„пјҢдҪ еҸҜд»ҘжҠҠжө·йҮҸдҝЎжҒҜеҺӢзј©жҲҗеҸҜжҺ§ж•°йҮҸзҡ„ж‘ҳиҰҒпјҢеҶҚеҲҶиҖҢжІ»д№ӢгҖӮ

иҝҷе°ұи§ЈеҶідәҶж–Үз« ејҖеӨҙжҸҗеҲ°зҡ„вҖқеӣҫд№Ұз®ЎзҗҶе‘ҳвҖқй—®йўҳвҖ”вҖ”GraphRAG дёҚйңҖиҰҒдёҖж¬ЎиҜ»е®Ңж•ҙжң¬д№ҰпјҢе®ғеҸӘйңҖиҰҒиҜ»жҜҸдёҖз« зҡ„ж‘ҳиҰҒпјҢз„¶еҗҺз»јеҗҲеҮәе…Ёд№Ұдё»ж—ЁгҖӮ

дёҺе…¶д»– RAG еҸҳдҪ“зҡ„жЁӘеҗ‘еҜ№жҜ”

GraphRAG е№¶дёҚжҳҜе”ҜдёҖзҡ„ RAG еўһејәж–№жЎҲгҖӮжҲ‘д»¬жқҘеҜ№жҜ”еҮ з§Қдё»жөҒеҸҳдҪ“пјҡ

ж–№жЎҲ ж ёеҝғжҖқи·Ҝ ж“…й•ҝ дёҚж“…й•ҝ
Naive RAG ж–ҮжЎЈеҲҶеқ— вҶ’ еҗ‘йҮҸеҢ– вҶ’ Top-K жЈҖзҙў зІҫзЎ®жҹҘжүҫгҖҒFAQ е…ЁеұҖжҖ§й—®йўҳгҖҒеӨҡи·іжҺЁзҗҶ
HyDE е…Ҳи®© LLM з”ҹжҲҗеҒҮи®ҫжҖ§зӯ”жЎҲпјҢеҶҚз”Ёзӯ”жЎҲеҒҡжЈҖзҙў й—®йўҳжЁЎзіҠж—¶жҸҗеҚҮеҸ¬еӣһ еўһеҠ дёҖж¬Ў LLM и°ғз”ЁжҲҗжң¬
Multi-hop RAG еӨҡиҪ®жЈҖзҙўпјҢжҜҸиҪ®еҹәдәҺдёҠиҪ®з»“жһңжү©еұ• еӨҡжӯҘжҺЁзҗҶй—®йўҳ е»¶иҝҹй«ҳгҖҒй”ҷиҜҜзҙҜз§Ҝ
GraphRAG жһ„е»әзҹҘиҜҶеӣҫи°ұ + зӨҫеҢәж‘ҳиҰҒ + Map-Reduce е…ЁеұҖжҖ§жҹҘиҜўгҖҒе…ізі»жҺЁзҗҶ зҙўеј•жҲҗжң¬й«ҳгҖҒдёҚйҖӮеҗҲйў‘з№Ғжӣҙж–°
LightRAG иҪ»йҮҸзә§еӣҫи°ұ RAGпјҢеҸҢеұӮзҙўеј•пјҲе®һдҪ“/е…ізі» + ж–ҮжЎЈпјү жҲҗжң¬ж•Ҹж„ҹеңәжҷҜ зӨҫеҢәеұӮзә§ж‘ҳиҰҒиғҪеҠӣејұ

GraphRAG зҡ„зӢ¬зү№д»·еҖјеңЁдәҺе…ЁеұҖжҗңзҙўиғҪеҠӣгҖӮе…¶д»–ж–№жЎҲжң¬иҙЁдёҠиҝҳжҳҜеңЁвҖқжүҫзүҮж®өвҖқпјҢеҸӘжңү GraphRAG еңЁзҙўеј•йҳ¶ж®өе°ұжҠҠе…ЁеұҖдҝЎжҒҜеҺӢзј©еҘҪдәҶгҖӮ

GraphRAG зҡ„д»Јд»·д№ҹеҫҲжҳҺжҳҫпјҡ

  1. зҙўеј•жҲҗжң¬й«ҳпјҡжҜҸдёӘж–ҮжЎЈеқ—йғҪиҰҒи°ғ LLM еҒҡе®һдҪ“жҸҗеҸ–пјҢдёҖдёӘ 1000 зҜҮж–ҮжЎЈзҡ„иҜӯж–ҷеә“еҸҜиғҪйңҖиҰҒж•°еҚғж¬Ў LLM и°ғз”Ё
  2. жӣҙж–°жҲҗжң¬й«ҳпјҡж–°еўһж–ҮжЎЈйңҖиҰҒйҮҚж–°жҸҗеҸ–е®һдҪ“гҖҒеҗҲе№¶еҲ°еӣҫи°ұгҖҒйҮҚж–°еҒҡзӨҫеҢәжЈҖжөӢ
  3. дёҚйҖӮеҗҲе®һж—¶еңәжҷҜпјҡзҙўеј•жһ„е»әеҸҜиғҪйңҖиҰҒж•°е°Ҹж—¶пјҢдёҚйҖӮеҗҲйңҖиҰҒеҚіж—¶жӣҙж–°зҡ„зҹҘиҜҶеә“

д»Җд№Ҳж—¶еҖҷиҜҘз”Ё GraphRAGпјҹ

  • дҪ зҡ„зҹҘиҜҶеә“зӣёеҜ№зЁіе®ҡпјҲдёҚжҳҜжҜҸеҲҶй’ҹйғҪеңЁеҸҳпјү
  • з”ЁжҲ·з»Ҹеёёй—®е…ЁеұҖжҖ§гҖҒжҖ»з»“жҖ§зҡ„й—®йўҳ
  • ж–ҮжЎЈд№Ӣй—ҙжңүдё°еҜҢзҡ„е®һдҪ“е…ізі»
  • дҪ жңүйў„з®—жүҝжӢ…зҙўеј•йҳ¶ж®өзҡ„ LLM и°ғз”ЁжҲҗжң¬

д»Җд№Ҳж—¶еҖҷдёҚиҜҘз”Ёпјҹ

  • зҹҘиҜҶеә“йў‘з№Ғжӣҙж–°пјҲиҖғиҷ‘дј з»ҹ RAG + еўһйҮҸзҙўеј•пјү
  • з”ЁжҲ·дё»иҰҒй—®зІҫзЎ®жҹҘжүҫзұ»й—®йўҳпјҲдј з»ҹ RAG е°ұеӨҹдәҶпјү
  • LLM и°ғз”Ёйў„з®—жңүйҷҗпјҲиҖғиҷ‘ LightRAG зӯүиҪ»йҮҸж–№жЎҲпјү

Spring Boot + LangChain4j е®һжҲҳ

зҗҶи®әи®ІеӨҹдәҶпјҢжқҘеҶҷд»Јз ҒгҖӮжҲ‘д»¬з”Ё LangChain4j е®һзҺ°дёҖдёӘз®ҖеҢ–зүҲзҡ„ GraphRAGпјҢеҢ…еҗ«е®һдҪ“жҸҗеҸ–гҖҒеӣҫи°ұжһ„е»әе’ҢеұҖйғЁжҗңзҙўгҖӮ

йЎ№зӣ®дҫқиө–

е®һдҪ“-е…ізі»жҸҗеҸ–еҷЁ

жәҗз Ғи§Јжһҗпјҡиҝҷж®өд»Јз Ғзҡ„ж ёеҝғжҳҜ prompt и®ҫи®ЎгҖӮжіЁж„ҸеҮ дёӘз»ҶиҠӮпјҡ

  1. жҳҺзЎ®жҢҮе®ҡдәҶе®һдҪ“зұ»еһӢе’Ңе…ізі»зұ»еһӢвҖ”вҖ”иҝҷжҜ”и®© LLM иҮӘз”ұеҸ‘жҢҘж•ҲжһңеҘҪеҫ—еӨҡпјҢеҮҸе°‘дәҶжӯ§д№үе’ҢдёҚдёҖиҮҙ
  2. иҰҒжұӮиҝ”еӣһ JSON ж јејҸвҖ”вҖ”й…ҚеҗҲ LangChain4j зҡ„з»“жһ„еҢ–иҫ“еҮәиғҪеҠӣпјҲеҸӮи§Ғ з»“жһ„еҢ–иҫ“еҮәе®һжҲҳпјүпјҢеҸҜд»ҘзӣҙжҺҘжҳ е°„еҲ° Java еҜ№иұЎ
  3. е®№й”ҷеӨ„зҗҶвҖ”вҖ”LLM иҝ”еӣһзҡ„ JSON з»ҸеёёеёҰжңү markdown д»Јз Ғеқ—ж Үи®°пјҢйңҖиҰҒжё…зҗҶ

зҹҘиҜҶеӣҫи°ұеӯҳеӮЁ

жңҖдҪіе®һи·өпјҡ

  2. еҲ«еҗҚиҜҚе…ёпјҡз»ҙжҠӨдёҖд»Ҫе®һдҪ“еҲ«еҗҚжҳ е°„иЎЁпјҢзЎ¬зј–з Ғй«ҳйў‘еҲ«еҗҚ
  3. еөҢе…ҘзӣёдјјеәҰж¶Ҳжӯ§пјҡеҜ№е®һдҪ“еҗҚеҒҡеҗ‘йҮҸеҢ–пјҢи®Ўз®—зӣёдјјеәҰпјҢи¶…иҝҮйҳҲеҖјзҡ„и§ҶдёәеҗҢдёҖе®һдҪ“
